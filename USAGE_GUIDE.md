# 爱贝壳内容同步助手 - 完整功能指南

## 🚀 新增功能（v1.1.0）

### 1️⃣ 自动内容提取

**支持的平台：**
- ✅ 微信公众号 (mp.weixin.qq.com)
- ✅ 知乎 (zhihu.com)
- ✅ CSDN (csdn.net)
- ✅ 掘金 (juejin.cn)
- ✅ 简书 (jianshu.com)
- ✅ 微博 (weibo.com)
- ✅ 小红书 (xiaohongshu.com)
- ✅ B站 (bilibili.com)
- ✅ 抖音 (douyin.com)

**提取内容包括：**
- 📝 标题和作者
- 📄 完整内容和纯文本
- 🖼️ 所有图片（最多20张）
- 🏷️ 标签和分类
- 👍 点赞、评论等数据
- 🔗 原文链接

### 2️⃣ 内容预览面板

**功能特性：**
- 显示提取的标题、作者、内容摘要
- 网格显示图片预览（9张）
- 点击图片查看原图
- 完整的 Markdown 内容查看
- 自动滚动到最新内容

**操作步骤：**
1. 打开支持的平台
2. 点击扩展弹窗中的「获取内容」按钮
3. 等待内容提取完成
4. 在预览面板中查看所有内容
5. 可点击图片查看大图

### 3️⃣ 账号自动识别

**工作原理：**
- 扩展自动检测当前浏览器标签页
- 识别已登录的社交媒体账号
- 在平台选择器中显示 ✓ 标记
- 自动勾选已登录的平台

**支持识别的账号：**
- 微信公众号
- 知乎
- CSDN
- 掘金
- 小红书
- 微博
- B站
- 抖音

**使用示例：**
```
当前页面：https://mp.weixin.qq.com/...
→ 自动识别「微信公众号」已登录
→ 弹窗中微信平台显示 ✓
→ 自动勾选微信
→ 一键选择其他平台后直接同步
```

### 4️⃣ 一键同步功能

**同步流程：**

1. **提取内容** - 从当前页面自动提取
2. **选择平台** - 勾选要同步的目标平台
3. **点击同步** - 启动同步流程
4. **自动处理** - 根据平台进行以下操作：
   - **当前页面同步**：自动填充并打开发布按钮
   - **其他平台**：打开新标签页并进行同步
5. **人工确认** - 检查内容后点击发布

**为什么需要人工确认？**
- 确保内容完整性
- 避免误发错误内容
- 符合平台政策要求
- 保证账号安全

### 常见问题

**Q: 点击「获取内容」没有反应？**
- A: 检查是否在支持的平台上
- 打开 F12 控制台，查看错误信息
- 确保页面完全加载
- 刷新页面重试

**Q: 为什么检测不到我的账号？**
- A: 扩展通过 URL 检测登录状态
- 确保当前标签页是已登录的平台
- 某些平台需要完全加载才能识别

**Q: 同步后内容格式混乱？**
- A: 不同平台支持不同的格式
- 使用纯文本和基础格式最兼容
- 在目标平台手动调整格式

**Q: 多次同步会重复发布？**
- A: 扩展不会自动点击发布
- 需要在每个平台手动确认并发布
- 这样做是为了防止误发

## 🎯 最佳实践

1. **总是检查内容预览** - 同步前确保内容完整正确

3. **使用基础格式** - 不同平台兼容性不同
4. **保留原文链接** - 便于跟踪和引用
5. **定期更新** - 确保获得最新功能
