# 文章同步助手 Chrome Extension

这是一个可本地安装的 Chrome Manifest V3 扩展，基于「爱贝壳内容同步助手」业务文档中的多平台分发场景，提供文章发布前的整理、适配和检查能力。

## 功能

- 输入文章标题、正文/Markdown、目标读者和核心关键词，一键生成多平台发布包。
- 面向微信公众号、知乎、小红书、头条号、掘金/CSDN、Medium 输出差异化标题、导语、标签和发布提示。
- 自动生成发布前检查清单，覆盖 Markdown 层级、配图、账号登录、人工预览与版权声明。
- 支持复制 Markdown 版发布包，便于粘贴到团队文档或再交给内容同步工具分发。
- 保留浏览器本地设置页，API Key 等敏感信息只存储在 `chrome.storage.local`。

## 安装到 Chrome

1. 打开 Chrome，进入 `chrome://extensions/`。
2. 打开右上角「开发者模式」。
3. 点击「加载已解压的扩展程序」。
4. 选择本仓库目录 `/workspace/chatgpt`。
5. 点击扩展图标，开始制作文章发布包。

## 使用说明

1. 填写文章标题，并粘贴正文或 Markdown。
2. 补充目标读者和核心关键词，例如 `自媒体人, 多平台分发, 内容同步`。
3. 勾选要适配的平台，点击「生成发布包」。
4. 检查每个平台的标题、导语、标签和注意事项。
5. 点击「复制结果」，把 Markdown 发布包保存到文档或交给后续同步流程。

## 设计依据

业务文档强调：多平台运营者需要把同一篇文章同步到微信公众号、知乎、小红书、头条号、掘金/CSDN、Medium 等平台，并在发布前人工预览确认。因此本扩展聚焦「发布前文章准备」：先把一篇原稿整理成可检查、可复制、可跨平台适配的发布包，再进入实际同步工具。

## 隐私与安全

扩展不会上传文章内容。设置页中的 API Key 仅保存到浏览器本地 `chrome.storage.local`，不会写入代码仓库。请不要把密钥提交到 Git。
